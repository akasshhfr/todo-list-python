# To-Do List CLI App

A command-line To-Do List application built using Python. The application allows users to manage daily tasks with CRUD operations and stores tasks using file handling.

## Features

- Add Tasks
- View Tasks
- Delete Tasks
- Save Tasks to File
- Load Saved Tasks

## Technologies Used

- Python
- File Handling

## Concepts Learned

- Functions
- Lists
- Loops
- CRUD Operations
- File Handling
